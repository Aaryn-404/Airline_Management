# Airline Reservation and Ticket Management System

## Overview

This project is a console-based airline management system developed in C++. It allows an airline company to manage its fleet, routes, flights, crew members, passengers, reservations, and ticket sales through an interactive user interface.

The application was designed using Object-Oriented Programming (OOP) principles, including inheritance, composition, aggregation, associations between classes, dynamic memory management, and data structures.

## Features

### Fleet Management

* Register aircraft.
* View aircraft information.
* Update aircraft data.
* Delete aircraft records.
* Assign aircraft categories and specifications.

### Route Management

* Create routes.
* View available routes.
* Modify route information.
* Delete routes.
* Manage flight duration, stopovers, and ticket prices.

### Flight Management

* Create and manage flights.
* Assign routes to flights.
* Assign aircraft to flights.
* Register departure and arrival information.
* Track available and occupied seats.
* Manage passenger lists.

### Passenger Management

* Register passengers.
* View passenger information.
* Update passenger data.
* Delete passenger records.

### Crew Management

* Register pilots and flight attendants.
* Manage pilot and co-pilot assignments.
* Manage cabin crew assignments.
* Update and remove crew members.

### Reservation and Ticket Sales

* Create reservations.
* Assign passengers to flights.
* Purchase tickets.
* Generate ticket information automatically.
* Prevent duplicate passenger registrations within the same flight.
